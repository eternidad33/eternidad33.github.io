Hugo博客搭建

Hugo博客搭建

本地部署

推荐视频：手把手教你从0开始搭建自己的个人博客 |第二种姿势 | hugo

  1. 下载Hugo，下载地址：GitHub

  2. 配置环境变量

  3. 创建自己的站点文件夹hugo new site myHugoSite

    生成的目录如下：

    PS C:\Users\Administrator\Desktop\myHogoSite> tree /f
文件夹 PATH 列表
卷序列号为 2A8A-01A5
C:.
│  config.toml
│
├─archetypes
│      default.md
│
├─content
├─data
├─layouts
├─static
└─themes

  4. 新建文章

    在content目录下生成博客hugo new 我的第一篇博客.md

    在content/posts目录下生成博客hugo new posts/我的第二篇博客.md

  5. 下载m10c主题并配置

部署到GitHub

  1. 创建远程仓库,仓库名必须为用户名.github.io
  2. 生成public文件夹hugo -t 主题名 -b https://用户名.github.io -D
  3. 然后就是一系列git操作
  4. 进入public，初始化git仓库
  5. 添加到暂存区
  6. 提交仓库
  7. 连接远程仓库
  8. 推送到远程仓库
